# 📚 Reports

Bu repo, haftalık olarak öğrendiğim teknik konuların kısa ve öz notlarını içermektedir. Her rapor, o haftanın konusunu kendi anladığım şekilde açıklamaktadır.

## 📁 İçerik

Her hafta yeni bir rapor eklenmektedir. Raporlar Markdown formatında yazılmış olup aşağıdaki konuları kapsamaktadır:

- Bilgisayar Mimarisi temelleri (CPU, RAM, Cache)
- Bilgisayar Bilimleri ve yazılım geliştirme konuları
- Computer Vision ve yapay zeka alanındaki kavramlar

## 📂 Klasör Yapısı

```
reports/
├── week1_bilgisayar_mimarisi.md
├── week2_...
└── ...
```

## 🗒️ Not

Raporlar her hafta düzenli olarak güncellenmektedir. Amaç ezber değil, kavramları gerçekten anlamak ve bunu yazıya dökmektir.
